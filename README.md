# slimy-day
![Screenshot](images/preview/screenshot_2023-09-20-200513.png)

A top-down shooter.

## controls

<kbd>space</kbd> shoot

<kbd>←</kbd> <kbd>w</kbd> move left 

<kbd>→</kbd> <kbd>a</kbd> move right

<kbd>↑</kbd> <kbd>s</kbd> move up

<kbd>↓</kbd> <kbd>d</kbd> move down

move mouse to aim
